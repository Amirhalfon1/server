Kaltura API

Service Name permission
Description Permission service lets you create and manage user permissions
Actions
Name Description
add Adds a new permission object to the account.
get Retrieves a permission object using its ID.
update Updates an existing permission object.
delete Deletes an existing permission object.
list Lists permission objects that are associated with an account. Blocked permissions are listed unless you use a filter to exclude them. Blocked permissions are listed unless you use a filter to exclude them.
getCurrentPermissions Retrieves a list of permissions that apply to the current KS.
