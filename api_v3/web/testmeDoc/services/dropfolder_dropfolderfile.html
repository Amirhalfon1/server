Kaltura API

Service Name dropfolder_dropfolderfile
Description DropFolderFile service lets you create and manage drop folder files
Actions
Name Description
add Allows you to add a new KalturaDropFolderFile object
get Retrieve a KalturaDropFolderFile object by ID
update Update an existing KalturaDropFolderFile object
delete Mark the KalturaDropFolderFile object as deleted
list List KalturaDropFolderFile objects
ignore Set the KalturaDropFolderFile status to ignore (KalturaDropFolderFileStatus::IGNORE)
