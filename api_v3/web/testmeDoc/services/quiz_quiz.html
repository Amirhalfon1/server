Kaltura API

Service Name quiz_quiz
Description Allows user to handle quizzes
Actions
Name Description
add Allows to add a quiz to an entry
update Allows to update a quiz
get Allows to get a quiz
list List quiz objects by filter and pager
serve creates a pdf from quiz object The Output type defines the file format in which the quiz will be generated Currently only PDF files are supported
getUrl sends a with an api request for pdf from quiz object
