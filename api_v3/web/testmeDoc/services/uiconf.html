Kaltura API

Service Name uiconf
Description UiConf service lets you create and manage your UIConfs for the various flash components This service is used by the KMC-ApplicationStudio
Actions
Name Description
add UIConf Add action allows you to add a UIConf to Kaltura DB
update Update an existing UIConf
get Retrieve a UIConf by id
delete Delete an existing UIConf
clone Clone an existing UIConf
listTemplates retrieve a list of available template UIConfs
list Retrieve a list of available UIConfs
getAvailableTypes Retrieve a list of all available versions by object type
