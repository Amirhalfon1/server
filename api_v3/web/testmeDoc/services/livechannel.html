Kaltura API

Service Name livechannel
Description Live Channel service lets you manage live channels
Actions
Name Description
add Adds new live channel.
get Get live channel by ID.
update Update live channel. Only the properties that were set will be updated.
delete Delete a live channel.
list List live channels by filter with paging support.
isLive Delivering the status of a live channel (on-air/offline)
appendRecording Append recorded video to live entry
registerMediaServer Register media server to live entry
unregisterMediaServer Unregister media server from live entry
validateRegisteredMediaServers Validates all registered media servers
