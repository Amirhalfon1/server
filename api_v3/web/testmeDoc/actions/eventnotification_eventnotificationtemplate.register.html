Kaltura API

register.register
Description:
Register to a queue from which event messages will be provided according to given template. Queue will be created if not already exists
Input Params
Name Type Description Required Default Value
notificationTemplateSystemName string Existing push notification template system name V
userParamsArray array User params V
Output Type
KalturaPushNotificationData
Example HTTP Hit
http://www.kaltura.com/api_v3/service/appToken/action/add
JSON object: 
				

[
	notificationTemplateSystemName: "value",
	userParamsArray: [{
		objectType: "KalturaEventNotificationParameter",
		key: "value",
		description: "value",
		value: {
			objectType: "KalturaStringValue",
			value: "value"
		}
	}]
]
