Kaltura API

appendRecording.appendRecording
Description:
Append recorded video to live entry
Input Params
Name Type Description Required Default Value
entryId string Live entry id V
assetId string Live asset id V
mediaServerIndex 1 V
resource KalturaDataCenterContentResource V
duration float in seconds V
isLastChunk bool Is this the last recorded chunk in the current session (i.e. following a stream stop event) 1
Output Type
KalturaLiveEntry
Example HTTP Hit
http://www.kaltura.com/api_v3/service/appToken/action/add
JSON object: 
				

[
	entryId: "value",
	assetId: "value",
	mediaServerIndex: "value",
	resource: {
		objectType: "KalturaDataCenterContentResource",

	},
	duration: value,
	isLastChunk: value
]
