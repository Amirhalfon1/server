Kaltura API

bulkUploadAdd.bulkUploadAdd
Description:
Add new bulk upload batch job Conversion profile id can be specified in the API or in the CSV file, the one in the CSV file will be stronger. If no conversion profile was specified, partner's default will be used
Input Params
Name Type Description Required Default Value
fileData file V
bulkUploadData KalturaBulkUploadJobData 1
bulkUploadEntryData KalturaBulkUploadEntryData 1
Output Type
KalturaBulkUpload
Example HTTP Hit
http://www.kaltura.com/api_v3/service/appToken/action/add
JSON object: 
				

[
	fileData: "value",
	bulkUploadData: {
		objectType: "KalturaBulkUploadJobData",
		fileName: "value",
		emailRecipients: "value",
		numOfErrorObjects: value
	},
	bulkUploadEntryData: {
		objectType: "KalturaBulkUploadEntryData",
		conversionProfileId: value
	}
]
