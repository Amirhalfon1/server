Kaltura API

search.search
Description:
Search for media in one of the supported media providers
Input Params
Name Type Description Required Default Value
search KalturaSearch A KalturaSearch object contains the search keywords, media provider and media type V
pager KalturaFilterPager 1
Output Type
KalturaSearchResultResponse
Example HTTP Hit
http://www.kaltura.com/api_v3/service/appToken/action/add
JSON object: 
				

[
	search: {
		objectType: "KalturaSearch",
		keyWords: "value",
		searchSource: value,
		mediaType: value,
		extraData: "value",
		authData: "value"
	},
	pager: {
		objectType: "KalturaFilterPager",
		pageSize: value,
		pageIndex: value
	}
]
