Kaltura API

query.query
Description:
report query action allows to get a analytics data for specific query dimensions, metrics and filters.
Input Params
Name Type Description Required Default Value
filter KalturaAnalyticsFilter the analytics query filter V
Output Type
KalturaReportResponse
Example HTTP Hit
http://www.kaltura.com/api_v3/service/appToken/action/add
JSON object: 
				

[
	filter: {
		objectType: "KalturaAnalyticsFilter",
		from_time: "value",
		to_time: "value",
		metrics: "value",
		utcOffset: value,
		dimensions: "value",
		filters: [{
			objectType: "KalturaReportFilter",
			dimension: "value",
			values: "value"
		}]
	}
]
